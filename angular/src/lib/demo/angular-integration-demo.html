bellhopos Angular Components

Learn how to integrate bellhopos components in your Angular application

1. Module Import

First, import the BellhoposAngularModule in your app.module.ts:

import { BellhoposAngularModule } from '@actabldesign/bellhop-angular'; @NgModule({ imports: [ BrowserModule, BellhoposAngularModule ], declarations: [AppComponent], bootstrap: [AppComponent] }) export class AppModule { }

2. Basic Dashboard Layout

Create a complete dashboard layout with sidebar, appbar, and content area:

<div class="app-layout"> <!-- Sidebar --> <aside class="app-sidebar"> <app-sidebar></app-sidebar> </aside> <!-- Main Content --> <main class="app-main"> <!-- Header/Appbar --> <header class="app-header"> <app-appbar [isMenuOpen]="!pageNavCollapsed" (menuToggle)="onPageNavToggle()"> </app-appbar> </header> <!-- Content Container --> <div class="bellhop-content-container"> <!-- Page Navigation --> <nav class="page-nav" [class.collapsed]="pageNavCollapsed"> <app-page-navigation></app-page-navigation> </nav> <!-- Content Area --> <section class="bellhop-content-area"> <div class="bellhop-content-wrapper-full"> <router-outlet></router-outlet> </div> </section> </div> </main> </div>

3. Component Usage Examples

Buttons

<!-- Primary Button --> <app-button [label]="'Save Changes'" [hierarchy]="'primary'" [size]="'md'" (onClick)="handleSave()"> </app-button> <!-- Secondary Button --> <app-button [label]="'Cancel'" [hierarchy]="'secondary'" (onClick)="handleCancel()"> </app-button>

Container with Content

<app-container [title]="'User Dashboard'"> <div class="content-grid"> <div class="stat-card"> <h3>Total Users</h3> <p>2,847</p> </div> <div class="stat-card"> <h3>Revenue</h3> <p>$12,459</p> </div> </div> </app-container>

Notifications

<!-- Success Notification --> <app-notification [title]="'Success!'" [message]="'Your changes have been saved successfully.'" [variant]="'success'" [dismissible]="true"> </app-notification> <!-- Warning Notification --> <app-notification [title]="'Warning'" [message]="'Please review your settings before proceeding.'" [variant]="'warning'"> </app-notification>

Badges and Status Indicators

<app-badge [text]="'Active'" [variant]="'success'"></app-badge> <app-badge [text]="'Pending'" [variant]="'warning'"></app-badge> <app-badge [text]="'Inactive'" [variant]="'error'"></app-badge>

Dropdown Menu

<app-dropdown [label]="'Actions'" [items]="dropdownItems" (itemSelected)="handleMenuAction($event)"> </app-dropdown>

Toggle Switch

<app-toggle [label]="'Enable Notifications'" [checked]="notificationsEnabled" (toggleChange)="onToggleChange($event)"> </app-toggle>

4. Component TypeScript Integration

Here's how to handle component events and data in your TypeScript component:

export class DashboardComponent { pageNavCollapsed = false; notificationsEnabled = true; dropdownItems = [ { label: 'Edit', value: 'edit', icon: 'edit' }, { label: 'Delete', value: 'delete', icon: 'delete' }, { label: 'Share', value: 'share', icon: 'share' } ]; onPageNavToggle(): void { this.pageNavCollapsed = !this.pageNavCollapsed; } handleSave(): void { console.log('Saving changes...'); // Your save logic here } handleCancel(): void { console.log('Cancelling changes...'); // Your cancel logic here } handleMenuAction(action: string): void { console.log('Menu action selected:', action); // Handle dropdown actions } onToggleChange(enabled: boolean): void { this.notificationsEnabled = enabled; console.log('Notifications enabled:', enabled); } }

5. Required CSS Import

Important: Don't forget to import the BellhopOS styles in your src/styles.css:

@import '@actabldesign/bellhopos-styles/src/index.css'; /* Optional: Add global styles for better layout */ * { box-sizing: border-box; } html, body { height: 100%; margin: 0; padding: 0; font-family: var(--font-inter); background-color: var(--color-white); }

6. App Layout Styles

Add these styles to your app.component.css for the dashboard layout:

.app-layout { display: flex; height: 100vh; overflow: hidden; } .app-sidebar { width: 280px; background-color: var(--color-white); border-right: 1px solid var(--color-neutral-200); } .app-main { flex: 1; display: flex; flex-direction: column; overflow: hidden; } .app-header { background-color: var(--color-white); border-bottom: 1px solid var(--color-neutral-200); height: 64px; } .page-nav { width: 280px; background-color: var(--color-neutral-50); border-right: 1px solid var(--color-neutral-200); transition: width 0.3s ease; } .page-nav.collapsed { width: 0; overflow: hidden; }

🚀 Quick Start

Follow these steps to get started with BellhopOS Angular components:

  1. Install the package: npm install @actabldesign/bellhop-angular
  2. Import the module in your app.module.ts
  3. Import the CSS styles in your styles.css
  4. Add the layout styles to your app.component.css
  5. Start using the components in your templates!
