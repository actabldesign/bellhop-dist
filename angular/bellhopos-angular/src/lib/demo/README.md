# BellhopOS Angular Library Demos

This directory contains interactive demo pages that showcase the BellhopOS Angular component library.

## Demo Files

### 📊 `dashboard-demo.html`

A complete dashboard application layout demonstrating:

- Full BellhopOS design system in action
- Complete sidebar + appbar + content layout
- All major UI components (buttons, badges, notifications, tables)
- Real dashboard-style content with stats, forms, and data tables
- Responsive design patterns
- Interactive elements and hover states

**How to view:** Open the HTML file directly in your browser - it's completely self-contained!

### ⚡ `angular-integration-demo.html`

A step-by-step integration guide showing:

- How to import and configure the library
- Code examples for each component
- TypeScript integration patterns
- Required CSS imports
- Layout structure examples
- Complete setup instructions

**Perfect for:** Developers who want to understand how to integrate BellhopOS components into their Angular applications.

## Quick Start

1. **Install the library:**

   ```bash
   npm install @actabldesign/bellhop-angular
   ```

2. **View the demos:**

   ```bash
   # Navigate to your node_modules
   cd node_modules/@actabldesign/bellhop-angular/src/lib/demo/

   # Open either demo in your browser
   open dashboard-demo.html
   open angular-integration-demo.html
   ```

3. **Follow the integration guide** in the `angular-integration-demo.html` to add BellhopOS to your project.
